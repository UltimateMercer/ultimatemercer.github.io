## 🚀 **Overview**

**Glyph** is a personal project dedicated to exploring and organizing the **Mass Effect** universe, one of the most iconic in science fiction video games. With an interface inspired by the franchise's futuristic design, the platform seeks to gather information about **characters, threats, historical events, species, and technologies** of the galaxy.

🔗 **Access:** [glyph.ultimatemercer.com](https://glyph.ultimatemercer.com/)

## 🎯 **Project Objective**

- Create a **knowledge repository** about **Mass Effect** with immersive design and experience.
- Develop a modern and responsive interface, using **Next.js, TailwindCSS, and ShadCN UI**.
- Explore a **futuristic visual style**, inspired by the holographic interfaces of the Citadel and Geth terminals.
- Provide an efficient search system to navigate through characters, events, and technologies.

## 🛠️ **Technologies Used**

| Technology          | Description                                 |
| ------------------- | ------------------------------------------- |
| **React + Next.js** | Project structure, SSR/SSG for optimization |
| **TailwindCSS**     | Fast and modular styling                    |
| **ShadCN UI**       | Accessible and customizable components      |
| **Contentlayer**    | Markdown data management                    |
| **Framer Motion**   | Smooth animations and interactions          |
| **Vercel**          | Optimized deployment and hosting            |

## 🖥️ **Interface and Design**

The UI of **Glyph** is heavily inspired by **Mass Effect** visual elements, combining **bluish tones and neons** with holographic effects. Highlights include:

- **Futuristic typography** → Use of geometric and technological fonts.
- **Interactive components** → Inspired by the Normandy interface and Alliance terminals.
- **Immersive dark mode** → Predominance of metallic tones and neon lighting.
- **Smooth animations** → Elements that appear and disappear like a holographic interface.

## 📌 **Key Features**

✅ **Knowledge Base** → Information on iconic characters, races, events, and locations.  
✅ **Interactive Timeline** → Chronological organization of the franchise's events.  
✅ **Futuristic Visual** → Design inspired by the game's holographic interfaces.  
✅ **Optimized Performance** → Built for fast loading and smooth experience.

## 🔜 **Next Steps**

✔ Improve navigation and category system.  
✔ Expand the database with more information from the Mass Effect universe.

**Glyph** is a project in constant evolution, created with passion for **Mass Effect** and futuristic design. It aims to be more than just a database, but an **interactive experience** for fans of the franchise. Currently, I am still working on adding the images in general.
